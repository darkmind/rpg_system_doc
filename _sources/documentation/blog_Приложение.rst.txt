Описание
========

  **U.R.S.A. characters** - приложение для генерации персонажей по ролевой системе U.R.S.A., их хранения, вкладывания очков опыта, управление спец параметрами и здоровьем.

  Ссылка на последнюю версию `U.R.S.A. characters
  <https://drive.google.com/file/d/0B6BmLJGy-3xKMVpvMFdFbDQyMEk/view?usp=sharing>`_

Версия 2.0
==========

  - Добавлено имя хроники.

  - Добавлен полный лист персонажа - имя, хроника, класс, мировоззрение, список богов если жрец, атрибуты, способности, дополнения, классовые способности.

  - Добавлена генерация персонажа - для нового персонажа.

  - Выбор класса влияет на классовые способности( Сферы, Дисциплины или Дары ) и на специальные ресурсы.

  - Выбор мировоззрения влияет на доступный список богов для Жрецов.

  - Выбор бога влияет на список доступных Дисциплин.

  - Возле каждого имени группы параметров( атрибуты, способности, etc )указано количество очков которые нужно распределить. Для выбора первичных/вторичных/третичных атрибутов/способностей количество очков указывается в [] скобках.

  - Назначение группы происходит когда вкладываешь хотя бы 1 поинт в группу. Сначала первичные, потом вторичные, потом третичные. Не обязательно все первичные очки распределять сразу.

    Пример: абсолютно новый персонаж у которого есть 7/5/3 очков для атрибутов.

    Если вложить 1 пункт в Манипулирование( Социальный атрибут ) - Социальные атрибуты будут считаться первичной группой для которой 7 очков.

    Дальше вкладываем 2 пункта в Интеллект( Ментальный атрибут ) - Ментальные становятся вторичной группой с 5 очками. 

    Дальше можно продолжить вкладывать в Социальные атрибуты и т.д.

    Тоже самое для Способностей.

  - Если кликнуть на количество пунктов больше чем доступно - увеличится на доступное количество.

    Пример: Манипулирование равно 1 и осталось 2 очка на распределение в группе. Если кликнуть на 5ую точку в Манипулировании атрибут поднимется до 3х т.к. больше нет свободных очков.

  - Возможности поменять выбор нет.

  - Возможности убрать 1ую точку из способности, дополнений, классовых способностей нет.

  - Как только все очки будут распределены появится диалоговое окно с предложением перейти с распределению свободных очков. Если ответить Да, то подписи возле параметров поменяются на Свободные очки - общее количество и стоимость повышения на 1.

  - Как только все Свободные очки будут распределены появится диалоговое окно с предложением завершить генерацию. Если ответить Да, то подписи исчезнут и в параметрах персонажа ничего поменять нельзя.

  Планы на будущее:

  1. Возможность убирать поставленный первый поинт.

  2. Добавить классовые ограничения - для Жреца сразу 1 пункт Религии и т.д.

  3. Добавить спец ресурсы в процесс генерации - покупать Силу Воли за Свободные очки.

  4. Добавить параметр Опыта и повышение параметров за него.

  5. Возможность скрывать/показывать параметры которые не качались, т.е. нулевые.

  6. Показывать классовые Способности.
